Multi-Agent debate 模拟不同智能体之间的多轮讨论场景，通常包括几个 solver 和一个 aggregator。 典型情况下，solver 生成并交换他们的答案，而 aggregator 收集并总结答案。

我们实现了 [EMNLP 2024](#emnlp-2024) 中的示例，其中两个 solver 智能体将按固定顺序讨论一个话题，根据先前的辩论历史表达他们的论点。 在每一轮中，主持人智能体将决定是否可以在当前轮获得最终的正确答案。

```````python
import asyncio
import os

from pydantic import Field, BaseModel

from agentscope.agent import ReActAgent
from agentscope.formatter import (
    DashScopeMultiAgentFormatter,
)
from agentscope.message import Msg
from agentscope.model import DashScopeChatModel
from agentscope.pipeline import MsgHub

# 准备一个话题
topic = "两个圆外切且没有相对滑动。圆A的半径是圆B半径的1/3。圆A绕圆B滚动一圈回到起点。圆A总共会旋转多少次？"


# 创建两个辩论者智能体，Alice 和 Bob，他们将讨论这个话题。
def create_solver_agent(name: str) -> ReActAgent:
    """获取一个解决者智能体。"""
    return ReActAgent(
        name=name,
        sys_prompt=f"你是一个名为 {name} 的辩论者。你好，欢迎来到"
        "辩论比赛。我们的目标是找到正确答案，因此你没有必要完全同意对方"
        f"的观点。辩论话题如下所述：{topic}",
        model=DashScopeChatModel(
            model_name="qwen-max",
            api_key=os.environ["DASHSCOPE_API_KEY"],
            stream=False,
        ),
        formatter=DashScopeMultiAgentFormatter(),
    )


alice, bob = [create_solver_agent(name) for name in ["Alice", "Bob"]]

# 创建主持人智能体
moderator = ReActAgent(
    name="Aggregator",
    sys_prompt=f"""你是一个主持人。将有两个辩论者参与辩论比赛。他们将就以下话题提出观点并进行讨论：
``````
{topic}
``````
在每轮讨论结束时，你将评估辩论是否结束，以及话题正确的答案。""",
    model=DashScopeChatModel(
        model_name="qwen-max",
        api_key=os.environ["DASHSCOPE_API_KEY"],
        stream=False,
    ),
    # 使用多智能体格式化器，因为主持人将接收来自多于用户和助手的消息
    formatter=DashScopeMultiAgentFormatter(),
)


# 主持人的结构化输出模型
class JudgeModel(BaseModel):
    """主持人的结构化输出模型。"""

    finished: bool = Field(description="辩论是否结束。")
    correct_answer: str | None = Field(
        description="辩论话题的正确答案，仅当辩论结束时提供该字段。否则保留为 None。",
        default=None,
    )


async def run_multiagent_debate() -> None:
    """运行多智能体辩论工作流。"""
    while True:
        # MsgHub 中参与者的回复消息将广播给所有参与者。
        async with MsgHub(participants=[alice, bob, moderator]):
            await alice(
                Msg(
                    "user",
                    "你是正方，请表达你的观点。",
                    "user",
                ),
            )
            await bob(
                Msg(
                    "user",
                    "你是反方。你不同意正方的观点。请表达你的观点和理由。",
                    "user",
                ),
            )

        # Alice 和 Bob 不需要知道主持人的消息，所以主持人在 MsgHub 外部调用。
        msg_judge = await moderator(
            Msg(
                "user",
                "现在你已经听到了他们的辩论，现在判断辩论是否结束，以及你能得到正确答案吗？",
                "user",
            ),
            structured_model=JudgeModel,
        )

        if msg_judge.metadata.get("finished"):
            print(
                "\n辩论结束，正确答案是：",
                msg_judge.metadata.get("correct_answer"),
            )
            break


asyncio.run(run_multiagent_debate())

```````

```
Alice: 正方认为，当圆A绕着圆B滚动一圈回到起点时，圆A总共会旋转3次。为了理解这一点，我们可以考虑圆周长的关系和圆A与圆B的相对运动。

首先，设圆B的半径为R，则圆A的半径为R/3。因此，圆B的周长是2πR，而圆A的周长是2π(R/3) = 2πR/3。当圆A绕着圆B无滑动地滚动一圈时，它所走过的距离等于圆B的周长，即2πR。

因为圆A每完成一次自转，就会走过一个自己周长的距离，也就是2πR/3。那么，在走过2πR的距离中，圆A需要完成的自转次数就是：

\[ \text{自转次数} = \frac{\text{圆B的周长}}{\text{圆A的周长}} = \frac{2\pi R}{2\pi R/3} = 3 \]

所以，圆A绕圆B滚动一圈回到起点的过程中，圆A总共会旋转3次。
Alice: 综上所述，正方的观点是：当圆A绕着圆B外切滚动一圈并且回到起点时，由于圆A的半径是圆B的1/3，圆A将会相对于自身轴心旋转3次。
Bob: 正方的观点基于一个直观但不完全准确的理解。实际上，当圆A绕着圆B外切滚动一圈并且回到起点时，我们不能简单地根据半径比例来判断圆A的自转次数。让我们考虑一下圆A在绕行过程中所经历的情况。

首先，圆A的周长是圆B周长的1/3，这意味着如果圆A只是沿着直线滚动而不绕圆B的话，它需要滚动3次自身的周长才能覆盖圆B的一圈周长。但是，当圆A绕着圆B滚动时，它不仅要滚动自己的周长，还要考虑到它同时围绕着圆B这个中心点进行公转。

当圆A完成一次完整的绕行回到起始位置时，它不仅完成了相对于地面的移动（即绕圆B的公转），还因为与圆B的接触而产生了额外的自转。这种情况下，圆A的自转次数会比简单的周长比较要多。

我们可以用一个物理模型来理解这一点：想象一根绳子固定在圆B上，并且缠绕在圆A上，当圆A绕着圆B滚动时，这根绳子会不断缠绕和解开。每次绳子完全解开，圆A就相对自身轴心旋转了一次。由于圆A绕圆B一周，相当于绳子被解开又重新缠绕了4次，因此圆A将会相对于自身轴心旋转4次，而不是3次。

所以我的观点是，圆A绕圆B外切滚动一圈并且回到起点时，圆A将会相对于自身轴心旋转4次。
Aggregator: 反方提出了一个更深入的分析，指出当圆A绕着圆B外切滚动一圈并且回到起点时，它不仅会因为自身的周长而旋转，还会因为围绕圆B公转而产生额外的自转。基于这个分析，圆A实际上会相对于自身轴心旋转4次。

为了进一步解释这一点，我们可以用一个数学公式来计算圆A的自转次数。设圆B的半径为R，那么圆A的半径就是R/3。圆A绕圆B一周的路径长度是圆B的周长，即2πR。对于圆A来说，它每走过的距离等于其周长（2π(R/3)）就会完成一次自转。但是，因为它同时在绕着圆B公转，所以总的自转次数将是路径长度除以圆A的周长加上1次公转带来的自转。

因此，圆A绕圆B一周的自转次数为：(2πR / (2π(R/3))) + 1 = 3 + 1 = 4次。

综上所述，正确的答案是圆A将会相对于自身轴心旋转4次。

辩论结束，正确答案是： 圆A绕圆B外切滚动一圈回到起点时，总共会旋转4次。

```
